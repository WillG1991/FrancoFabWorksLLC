-----------------------------------------
            Franco Fab Works LLC Website
-----------------------------------------

Overview
--------
This project is the source code for the Franco Fab Works LLC website, a professional web presence showcasing the services, portfolio, and contact information for Franco Fab Works LLC, a company specializing in welding and fabrication.

Features
--------
- Responsive Design: The website is designed to be responsive, ensuring a seamless experience across various devices and screen sizes.
- Portfolio Section: Highlighting examples of craftsmanship in welding and fabrication.
- Contact Form: An integrated form for easy communication.
- Dynamic Hero Section: A hero section with interchangeable video and GIF for different screen sizes.

Technologies Used
-----------------
- HTML5: For structuring the web content.
- CSS3: For styling the website.
- JavaScript: For interactive elements.
- Bootstrap: A framework used for responsive design.

Usage
-----
To view the website:
1. Clone the repository to your local machine.
2. Open the 'index.html' file in a web browser.
